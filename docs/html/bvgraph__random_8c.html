implementation of routines for bvgraph with random access

Implement the set of routines to work with the bvgraph with random access, but only when offset_step = 1

Access the successors of a vertex.

This operation is not thread-safe and heavily modifies the random access iterator. (Don't worry though, it doesn't touch the underlying graph, so please do use one random iterator for each thread in your code.)

Parameters
[in]ria random access iterator for the graph
[in]xthe index of the node (i in [0,g->n-1])
